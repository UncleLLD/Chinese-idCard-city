# Chinese-idCard-city
Chinese ID card is related to the city and area
验证中国身份证, 前4,6位对应市区

## Dirs
* `idcard-4-city.txt`: map first 4 idcard numbers into city level. sep is '\t'
	- 1101	北京市
	- except:
		- 429004	仙桃市
		- 429005	潜江市
		- 429006	天门市
		- 429021	神农架林区
* `idcard-6-city-area.txt`: map first 6 idcard numbers into city and area level.
	- 110101	北京市东城区
